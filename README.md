# RYSurvival-Tools
任渊生存服务端工具  
包含3个模式: 测试模式 打包模式 帮助模式
使用Batch编写,不支持linux  

# 打包模式
用于方便快速的打包并拟定版本号  
并且在切换主仓库分支后不需要手动修改文件  
  
# 测试模式
用于在不影响服务端文件的情况下测试服务端  
并且在切换主仓库分支后不需要手动修改文件  

# 帮助模式
用于获取此工具的使用方式    
  
# 使用方式
使用Github Desktop  
分别clone任渊生存主仓库和此工具  
打开此工具所在的本地目录  
将任渊生存主仓库切换到你想要测试的分支  
找到tools文件夹,打开你想要使用的工具即可
